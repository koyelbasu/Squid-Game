# 🎮 Squid Game — GitHub Repository Structure
## 🔥 Project Overview

Squid Game Landing Page is a cinematic and modern frontend website inspired by the popular Netflix series Squid Game.
The landing page features bold visuals, dark red gradients, neon aesthetics, interactive sections, and a fully responsive UI built using pure HTML5 and CSS3.

## ✨ Key Highlights
🎮 Squid Game inspired modern UI<br>
🔥 Dark cinematic red gradient theme<br>
📱 Fully responsive landing page<br>
🎨 Stylish typography & glowing effects<br>
⚡ Smooth hover animations<br>
🖼️ High-quality character showcase section<br>
🚀 Clean and organized frontend structure<br>
💻 Beginner-friendly codebase
## 🛠️ Tech Stack
🌐 HTML5
🎨 CSS3
